This paper introduces FIRESPARQL, a modular framework for translating natural language questions into SPARQL queries over scholarly knowledge graphs. The framework combines fine-tuned LLMs with optional retrieval-augmented generation and a SPARQL query correction layer. Experiments on the SciQA Benchmark compare zero-shot, one-shot, RAG-enhanced, and fine-tuned configurations, showing that fine-tuning achieves the strongest performance for both query accuracy and query result accuracy.
